点击 C#课程设计\爬虫\bin\Debug 目录下的爬虫.exe 即可使用

优酷的部分爬的是《都挺好》第一集的弹幕，填分钟数就好了，因为单线程如果循环爬取集数演示时间太久了。。

若需要使用图表的话，需要将数据库文件导入数据库才可以使用

注意：大部分功能都需要在有网络的情况下使用，否则可能会出现使用不了的情况。
